withQuery(selector) → {Domler}

Navigate into a child element by a CSS selector.

Works exactly like with, except the navigation target is resolved using querySelector instead of a property path.

Use without to step back out of the selected element, just as with property-based navigation.

Parameters:
Name Type Description
selector string

CSS selector.
Returns:

this

Type
Domler
Example
// CSS selector navigation:
handler.withQuery("#container > span.item");

// Step back to the parent context:
handler.without();
