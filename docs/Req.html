(static) enforceMethod(methodOrMethods, req, res) → {boolean}

Enforce allowed HTTP methods on a request.

Accepts a single method, multiple comma/space-separated string, or array of methods. If the request method is not allowed, responds with 405 Method Not Allowed and sets the Allow header.

Examples:

Outpost.enforceMethod('GET', req, res);
Outpost.enforceMethod('GET,POST', req, res);
Outpost.enforceMethod(['GET', 'POST', 'PUT'], req, res);
Parameters:
Name Type Description
methodOrMethods string | Array.<string>

Allowed HTTP method(s)
req http.IncomingMessage
res http.ServerResponse
Returns:

True if method allowed, false if disallowed
