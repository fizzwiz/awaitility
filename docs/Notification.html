Class: Notification

Notification(msg, payloadopt, emitteropt, sourceopt)

A serializable base class for notifications, such as Errors, Messages, and Events.

Can be used locally or across distributed systems. In inter-machine communication, a Notification is never treated as a valid result: it can either be emitted as an event or transformed into an error.
